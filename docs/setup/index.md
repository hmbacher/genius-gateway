# Installation & Setup

Complete installation and setup guide for the ESP32 Genius Gateway.

## Overview

This section covers everything you need to get your Genius Gateway up and running:

- Hardware setup and assembly
- Firmware installation and configuration  
- Network connectivity and WiFi setup
- MQTT integration with home automation systems

## Quick Navigation

- **[Hardware Setup](hardware.md)** - Physical assembly and wiring
- **[Firmware Installation](firmware.md)** - Flashing and initial setup

## Prerequisites

Before starting, ensure you have:

- ESP32-S3 development board or complete gateway PCB
- CC1101 radio module (if using development board)
- 868 MHz antenna
- USB-C cable for programming and power
- Computer with internet connection for setup
